# 🏋️ MEMBERSHIP GYM SYSTEM

Sistem manajemen gym lengkap dengan mobile app, admin web, dan backend API.

## 📚 Dokumentasi Lengkap

**Baca dokumentasi lengkap di:** [`DOKUMENTASI.md`](./DOKUMENTASI.md)

## 🚀 Quick Start

### 1. Setup Database
```bash
# Jalankan Laragon (MySQL + Apache)
# Buka phpMyAdmin dan import: api/database/membership_gym.sql
```

### 2. Jalankan Backend
```bash
cd api
npm install
npm start
```

### 3. Buka Admin Web
```
Buka file: admin_web/index.html di browser
Login: admin@gym.com / admin123
```

### 4. Jalankan Mobile App
```bash
flutter pub get
flutter run
```

## 🔧 Konfigurasi

**✅ Saat ini sudah dikonfigurasi untuk localhost**

- Backend API: `http://localhost:3000`
- Flutter App: localhost mode aktif
- Admin Web: localhost mode aktif

## 📱 Komponen

| Komponen | Teknologi | Port |
|----------|-----------|------|
| Backend API | Node.js + Express | 3000 |
| Admin Web | HTML/CSS/JS | - |
| Mobile App | Flutter | - |
| Database | MySQL (Laragon) | 3306 |
