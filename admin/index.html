Teknologi NFC IoT

Check-in Tanpa
Antrian & Kerumitan

Sistem NFC GymKu mendukung dua mode: ACR122U USB Reader untuk meja admin dan HCE Android agar HP member berfungsi sebagai kartu virtual — tanpa cetak kartu fisik!

🛡️
Proteksi Double Check-in 3 Lapis

Bridge cooldown 5 detik, Browser debounce 10 detik, Atomic SQL INSERT WHERE NOT EXISTS.

📱
HP sebagai Kartu Virtual (HCE)

Flutter app mengimplementasi Host Card Emulation — tap HP ke reader, langsung check-in!

🔒
Proteksi Berbasis Langganan

Check-in otomatis diblokir jika membership member sudah expired.

💳
Kartu Fisik NFC

Dukung NTAG213 & Mifare Classic 1K. Admin bisa program kartu fisik langsung dari app.
